# 移动工具(v)

移动工具可以移动图层。

选中自动选择后，会自动选择当前位置有内容的图层。

# 钢笔工具(p)

钢笔工具是用来绘制路径的工具。

# 图形工具(u)

图形工具可以绘制常见的图形，包括矩形，椭圆型，多边形，圆角矩形和自定义图形。

按住shift绘制可以绘制正图形（正方形，正圆形）

# 渐变工具(g)

在图层上绘制渐变的颜色

选中渐变工具可以设置渐变条

  • 上方的方块可以设置透明度，下方的方块可以设置颜色
  • 两个方块中间的小三角可以决定这种颜色的渐变占比

在图层上进行绘制时，拖拽的方向和长度可以决定渐变的方向和渐变区域的大小。

# 文字工具(t)

文字工具可以创建文字图层，用于展示文字

文字有很多属性可以设置，包括字体，文字宽度，字号，文字颜色，背景颜色，行间距，字间距

文字图层分为点文字和块文字

点文字不会自动换行，使用文字工具在图层上点击就可以。

块文字需要在文字工具下拖拽，会自动换行，当文字显示不全时会出现提示。
