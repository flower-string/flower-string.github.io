# 混合模式

CSS也有混合模式了，这么搞下去感觉CSS需要一个可视化编辑器了。 从动画到渐变到阴影到滤镜再到混合模式，快赶上PS了。

# 属性

css的混合模式有两个相关的属性

  • background-blend-mode： 混合当前元素的所有背景
  • mix-blend-mode：与其他元素的背景进行混合

# 模式

CSS的混合模式有16种，常用的有5种 效果如下： ![[Pasted image 20230923204123.png]]![[Pasted image 20230923204153.png]] CSS混合模式的内容很好，主要是对16种混合模式的理解程度。 熟练使用混合模式配合想象力能用创作出奇妙的效果。
